HeldenWeb Export

Ein Plugin für die Helden-Software, mit dem Helden mit dem HeldenWeb synchronisiert werden können.

HeldenWeb Export ist ein Plugin für die Helden-Software. Mit der Helden-Software können Helden für das Rollenspiel Das Schwarze Auge (DSA) erstellt und verwaltet werden. Das HeldenWeb Export Plugin erlaubt es, diese Helden in das HeldenWeb zu exportieren.

Das neueste Build-Ergebnis kann hier heruntergeladen werden und entspricht dem aktuellen Stand des Repositories. Diese Version ist nicht unbedingt stabil und kann Fehler enthalten. Das aktuelle Release steht auf der Download-Seite des Projektes zur Verfügung.
